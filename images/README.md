# Billboard photos

Drop site photos here, then set the `photo` field in `js/locations.js`.

Naming convention: `<location-id>-<n>.jpg`

- kijenge-1.jpg, kijenge-2.jpg
- matunda-1.jpg
- carshowroom-1.jpg, carshowroom-2.jpg
- summit-1.jpg
- tengeru-1.jpg
- total-njiro-1.jpg

Landscape 16:10 or wider looks best in the cards. Keep files under ~400KB (export at ~1600px wide).
